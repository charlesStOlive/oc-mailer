### Variable environement pour Mailer
L'utilisation des mails demande la configuration duivante : 
```
# Codes de connection sur MJML
MJML_API_ID=xxx
MJML_API_SECRET=xxx

# Codes de connection sur mailgun
MAIL_DRIVER=smtp
MAIL_HOST=smtp.mailgun.org
MAIL_PORT=587
MAIL_ENCRYPTION=tls
```
